In questa cartella sono presenti i file relativi all'ontologia utilizzata. In particolare:
- PCSCOPRO_DATA.owl è l'ontologia popolata.
- PCSCOPRO_DATA_RDF.owl è l'ontologia popolata in formato RDF.
- PCSCOPRO_DATA_SWRL.owl è l'ontologia popolata e contiene anche i fatti inferiti dal reasoner, inclusi quelli derivanti dalle regole SWRL implementate.
- PCSCOPRO.bak è l'ontologia NON popolata.
- Dati contiene tutti i dataset trovati che potevano essere utili al popolamento dell'ontologia.
- Dati utilizzati contiene i dati utilizzati per il popolamento dell'ontologia e le regole di import per Cellfie.
- Ontologie di supporto contiene tutte le ontologie trovate nella fase iniziale del progetto.
- Ontologie utilizzate contiene le ontologie utilizzate per il progetto.
- Annotazioni contiene un file csv che contiene tutte le annotazioni effettuate con il BMPNSemanticAnnotator, un file xlsx che contiene le descrizioni delle annotazioni effettuate, le ontologie generate dall'annotatore. Inoltre contiene una cartella Controprova dove ci sono gli stessi file modificati per fare la controprova delle regole implementate.
