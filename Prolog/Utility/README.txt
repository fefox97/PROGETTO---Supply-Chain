Qui sono contenute tutti i file Prolog di supporto, tra cui 
- regoleSupportoBPMNAggiustate.pl che contiene le regole del Prof. Esposito aggiustate da me.
- regoleSupportoBPMNAggiustateOriginali.pl che contiene le regole del Prof. Esposito originali.
- solve_path.py che contiene uno script Python per la correzione delle stringe malformate nei fatti Prolog ottenuti dalla conversione dell'ontologia annotata con l'annotatore.
- MM_Ontology.owl e BPMN2_Ontology.owl sono delle ontologie di supporto.
- thea-master contiene la libreria di thea.
